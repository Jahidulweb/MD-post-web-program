# কীভাবে একটা একটা GitHub স্টাইলের Markdown পোস্ট সিস্টেম বানাবে?
##### নিচের ধাপগুলো অনুসরণ করো
### Step -1
নিজের ওয়েবসাইটের রুট ফোল্ডারে `post.html` নামে একটা HTML ফাইল রাখো এবং তার ভিতরে এই রেপোর post.html এর মধ্যে থাকা কোড কপি করে নিয়ে পেস্ট করো।

### Step -2
posts নামের একটা ফোল্ডার বানাও সেখানে তোমার সব `.md` পোস্ট আপলোড করো। টেস্ট করার জন্য `test.md` নামে একটা ফাইল বানাও এবং তার ভিতরে কিছু লিখে সেটা posts ফোল্ডারের মধ্যে রাখো।

সার্ভার চালিয়ে ব্রাউজারে লিখ
```
http://localhost:8080/post.html?file=test.md
```
তুমি তোমার .md ফাইলটির একদম GitHub স্টাইলের প্রিভিউ দেখতে পাবে।
